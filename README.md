# ClassiSphere Web Client

ClassiCube-based web client. Works on Android 6+, iOS 12+, desktop.

## Play
- [Play Now](play.html)

## Multiplayer
Edit arguments in play.html:
```
arguments: ['YourName', '', 'server.address.com', '25565']
```

## Local Testing
```
python -m http.server 8000
```
Open http://localhost:8000/play.html
